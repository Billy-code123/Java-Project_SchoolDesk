# Java-Project_SchoolDesk
SchoolDesk desktop app made with Java
# Application Development Assignment

This is a Java Swing desktop application developed for the Application Development module.

The system allows the user to manage teachers, courses and results using a MySQL database.

## Technologies Used

- Java
- Java Swing
- MySQL
- JDBC
- Eclipse IDE

## Features

- Add, update and delete teachers
- Manage courses
- Store and retrieve data from a MySQL database
- Display results through Java Swing forms

## Database Setup

1. Open MySQL Workbench.
2. Create a new schema.
3. Import the SQL file from the `database` folder.
4. Update the database connection details in `DBConnection.java` if needed.

## How to Run

1. Open the project in Eclipse.
2. Make sure MySQL Server is running.
3. Import the database from `database/assignment_2026_database.sql`.
4. Add the MySQL Connector/J library to the project.
5. Run `Introduction_Form.java`.
